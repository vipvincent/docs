# 鞘翅胸甲

:::tip

需要使用1.21.2/1.21.3或以上版本

:::

胸甲與鞘翅可以透過鍛造台合成在一起，並同時裝備在身上使用，且兩者功能都能使用。

- 支持原版Minecraft的所有胸甲
- 不需資源包即可支持所有語言(需為Minecraft本身受支持的語言)

---
## 合成

:::info

胸甲的既有附魔、耐久度以及命名合成時會保留，但是鞘翅上的不會保留。

:::
