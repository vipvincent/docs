:::tip

模組與資料包的所有功能皆為相同。   
資料包僅會套用於單一世界，區網世界和伺服器都可以使用。   
模組會自動套用於所有世界，因此建議使用在伺服器。    

:::

:::info

模組載入器支持**Fabric**、**Forge**、**Neoforge**和**Quilt**  
Fabric需要[**Fabric API**](https://modrinth.com/mod/fabric-api)，Quilt需要[**Quilted Fabric API**](https://modrinth.com/mod/qsl)

:::
