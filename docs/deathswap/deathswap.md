# 死亡交換

死亡交換是一款遊戲，隊伍之間距離很遠，於30-200秒互相隨機交換，隊伍之間並沒有固定順序交換，而玩家唯一要做的事，就是設計陷阱，已讓下一個傳送到那裏的玩家致死，並且成為最後一個存活的隊伍。
